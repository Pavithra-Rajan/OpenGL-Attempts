# OpenGL-Attempts

To install OpenGL run the ```install.sh``` script.

### To run an OpenGL C++ file, run the following command
```console
 g++ <name of CPP file> -lglut -lGL -lGLEW -lGLU -o <name of executable>
./<executable file name>
```
